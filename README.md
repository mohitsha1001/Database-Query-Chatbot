# Database-Query-Chatbot

1. Developed the complete backend using Node.js and Express.js to handle user queries and communicate with OpenAI and Supabase.
2. Designed the chat interface using HTML, CSS, and JavaScript to enable seamless user interaction.
3. Integrated GPT-3.5 Turbo API to convert natural language inputs into SQL queries dynamically.
4. Implemented secure data processing and optimized query response formats for clear and readable outputs.
5. Added logging and error-handling mechanisms to maintain query records and improve troubleshooting.
